# War Card Game 

Simple school project made with HTML, CSS and JavaScript.
Link: https://vojna-cardgame.netlify.app/

This project is a card game called **War**.  
Cards are loaded from a REST API using **fetch()**.

## Features
- Get cards from API
- Compare two cards
- Show winner of round
- Simple game design

## Technologies
- HTML
- CSS
- JavaScript
- REST API
- Fetch

## Purpose
This project was created for school practice to learn:
- JavaScript
- Fetch API
- Working with REST API
- DOM manipulation
